# Change: 相机控制系统与输入封装系统，并实现 WASD/滚轮相机控制

## Why
当前项目已有 ECS 核心与 3D Demo，但缺少统一的相机控制接口和输入抽象层，无法方便地实现“用 WASD 移动相机、滚轮缩放”等编辑/玩法需求。需要先构建相机控制与输入封装两套能力，再基于二者实现具体的相机移动逻辑。

## What Changes
- 新增 **相机控制系统**：提供控制相机移动、缩放（或距离）的接口，可绑定到 Laya 场景中的 Camera 节点。
- 新增 **输入封装系统**：对键盘、鼠标等输入进行抽象，提供按键状态与滚轮增量等查询接口，便于后续替换或扩展输入源。
- 基于上述系统 **实现相机移动逻辑**：使用 WASD 控制相机平移、鼠标滚轮控制相机缩放（或前后距离），键位与灵敏度可配置。

## Impact
- **Affected specs（新增能力）**
  - `camera-control`：相机控制接口与基于输入的相机移动逻辑。
  - `input-abstraction`：键盘按键状态、鼠标滚轮等输入封装。
- **Affected code（预期）**
  - 新增 `src/input/` 或等价目录：输入抽象层实现（可依赖 Laya 键盘/鼠标 API）。
  - 新增 `src/camera/` 或 `src/game/camera/`：相机控制接口及 WASD/滚轮驱动逻辑。
  - `Main.ts` 或 Demo 入口：注册/挂接相机控制与输入更新。
